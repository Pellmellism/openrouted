Frequently Asked Questions

The plugin isn't generating alt tags for my images. What's wrong?
First, check if you've entered a valid OpenRouter API key in the settings. Next, ensure your server can make outbound API calls. If problems persist, check the plugin's logs or contact support.
How many images can I process per day?
OpenRouted uses OpenRouter's free tier, which typically allows for around 20-30 image generations per day. The exact number may vary based on OpenRouter's current quota system.
Can I customize how the AI describes my images?
Yes! In the plugin settings, you can provide custom instructions to guide the AI in generating alt text that matches your site's tone and style.
Is this plugin compatible with my theme/page builder?
OpenRouted is designed to work with standard WordPress media management. It should work with most themes and popular page builders, but specific compatibility may vary.
